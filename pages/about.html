About This Site

The website may take up to a minute to connect to the backend server during the initial load. Tokens expire after 10 minutes, so you'll need to log in again to generate new ones.

  • MongoDB Atlas: A cloud-based database for efficient data storage and management.
  • Express.js: A backend framework for handling server-side requests and routing.
  • JWT Authentication: Ensures secure user login and session handling.
  • Bcrypt: Encrypts user passwords for added security.
  • Node.js: A JavaScript runtime environment for building scalable backend applications.
